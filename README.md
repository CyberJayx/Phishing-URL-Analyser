# Phishing URL Reputation Checker

## Overview
[span_6](start_span)Inspired by my work in **Malware Triage** and **Phishing Analysis**[span_6](end_span), this script analyzes URLs for common indicators of compromise (IOCs) often found in phishing campaigns.

## Features
- **Keyword Detection:** Flags URLs containing sensitive terms like "login" or "secure."
- **Syntax Analysis:** Detects unusual characters (e.g., '@' or '-') used to mask malicious links.

- Threat Intelligence, IOC Correlation, Phishing Mitigation.
